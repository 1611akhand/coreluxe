# Resort Booking Website

## 📌 Project Overview
This is a responsive Resort Booking Website where users can explore resort details, view images, and check available services. The website is designed to provide a smooth and attractive user experience.

## 🚀 Features
- Modern and responsive UI design
- Resort listing with images
- User-friendly navigation
- Mobile-friendly layout
- Fast loading using Netlify deployment

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Netlify (for deployment)

## 🌐 Live Demo
https://resort-site-11.netlify.app/

## 📂 Installation & Setup
1. Clone the repository
2. Open the project folder
3. Run index.html in browser
